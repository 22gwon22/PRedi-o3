Predição de Rotas CCO - versão estática para Vercel

Esta versão NÃO TEM package.json, NÃO TEM Tailwind e NÃO precisa de npm install.

Como subir:
1. Extraia o ZIP.
2. Suba SOMENTE os arquivos index.html e README.txt para o GitHub.
3. Na Vercel, importe o repositório.
4. Deixe Build Command vazio e Output Directory vazio ou ./

Se o repositório antigo ainda tiver package.json, apague ele antes.
